# 2c_cache_mover
Приложение переносит стандартные каталоги кэша 1С из каталога профайла пользователя в указанное место.
При этом на в профайле создаются ссылки на новое расположение каталогов кэша.
Так как производится пренос существующих каталогов, сожержимое кэшей не теряется.

Ссылки создаются через программу junction64 из пакета sysinternals

В каталоге, в который будут переноситься кэши 1С создаются структура 
 dir\username\AppData\Local\1C\1cv8
 dir\username\AppData\Roaming\1C\1cv8


## формат

1c_profile_mover.ps1 [ -username  username] -dist path_to_folder
где
 -username  username - пользователь, кэш которого переносится, необязательный параметр, по умолчанию текущий пользователь
 -dist path_to_folder каталог, в котором будет создаваться каталоги кэша, обязательный параметр
